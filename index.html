A safety-focused patrolling robot using Raspberry Pi, integrated with AI (OpenCV) and multiple sensors (ultrasonic, gas, thermal) and webcam for live streaming and real-time monitoring and obstacle avoidance in hazardous environments. Gesture-Controlled Robot

Designed and built a gesture-controlled car using Arduino Uno as the main controller. Hand gestures are captured through an accelerometer (e.g., MPU6050), processed by Arduino, and transmitted wirelessly (using RF or Bluetooth modules) to control the car's movement—forward, backward, left, and right. This project demonstrates wireless communication, sensor integration, and real-time control, offering an interactive approach to robotic vehicle navigation.

4 DOF Robotic Arm

Developed a 4 Degrees of Freedom robotic arm using Arduino Uno, servo motors, and potentiometers for manual control. Each joint (base, shoulder, elbow, and wrist) is controlled via a dedicated potentiometer, enabling precise joint movement. The end effector (gripper) is operated by an additional servo for object gripping and release. This setup successfully performs basic pick-and-place operations, demonstrating practical implementation of robotic kinematics and manual control systems.
